#  F조 영단어 학습 플랫폼

누구나 쉽게 영단어를 암기하고 테스트할 수 있는 웹 서비스입니다.

##  주요 기능
* **로그인/ 로그아웃**: 나만의 단어장을 만들고 관리할 수 있습니다.
* **회원가입 기능**: 틀린 단어만 모아서 다시 복습합니다.
* **단어 학습장 기능**: 객관식/주관식 테스트로 실력을 점검합니다.
* **단어 퀴즈 기능**: 틀린 단어만 모아서 다시 복습합니다.
* **랭킹 리더보드 기능**: 틀린 단어만 모아서 다시 복습합니다.
* **학습 통계 대시보드 기능**: 틀린 단어만 모아서 다시 복습합니다.

##  기술 스택
* Language: Java 8+
* Framework: Spring Framework 5, Spring Security 
* Database: MySQL (또는 Oracle), MyBatis
* Server: Apache Tomcat 9.0 
* Frontend: JSP, JSTL
* Tool: Eclipse, Maven

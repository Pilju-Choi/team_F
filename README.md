#  F조 영단어 학습 플랫폼

누구나 쉽게 영단어를 암기하고 테스트할 수 있는 웹 서비스입니다.

##  주요 기능
* **로그인/ 로그아웃**: 로그인, 로그아웃 기능을 제공합니다.
* **회원가입 기능**: 회원가입 기능을 제공합니.
* **단어 학습장 기능**: 영어 닌이도에 맞는 영어 단어를 일차별로 제공한다..
* **단어 퀴즈 기능**: 저장된 단어를 기반으로 객관식 퀴즈를 풀며 암기 여부를 확실하게 점검합니다.
* **랭킹 리더보드 기능**: 퀴즈 점수를 바탕으로 사용자 간 순위를 매겨 학습에 대한 흥미와 경쟁심을 유발합니다.
* **학습 통계 대시보드 기능**: 시각화된 그래프로 제공하여 성장 과정을 한눈에 확인합니다.

##  기술 스택
* Language: Java 8+
* Framework: Spring Framework 5, Spring Security 
* Database: MySQL (또는 Oracle), MyBatis
* Server: Apache Tomcat 9.0 
* Frontend: JSP, JSTL
* Tool: Eclipse, Maven
